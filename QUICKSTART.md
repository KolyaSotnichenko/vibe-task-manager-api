# QUICKSTART

Коротка інструкція для запуску бекенд-додатку локально.

## Вимоги

- Node.js >= 18
- npm (йде разом з Node.js)

## Встановлення

```bash
npm install
```

## Змінні середовища

На даний момент обов'язкові змінні середовища відсутні.

За замовчуванням сервер стартує на порту `3000` (зашито в коді).
Якщо потрібно змінювати порт або інші налаштування — їх слід додати в код конфігурації.

## Запуск у dev-режимі

```bash
npm run start:dev
```

Після запуску API буде доступне за адресою:

```
http://localhost:3000
```

## Збірка та запуск у production-режимі

```bash
npm run build
npm run start
```

## Корисні команди

- `npm run lint` — перевірка eslint
- `npm test` — запуск тестів
- `npm run openapi:export` — експорт OpenAPI специфікації
